# Play Logo - Three.js WebGPU

Two versions available:
- `index.html` + `main.js` - TSL version (recommended)
- `wgsl.html` + `main-wgsl.js` - WGSL version

Both use WebGPU with fancy shaders for a volumetric play button logo.

Run with:
```
python3 -m http.server 8080
```

Then open http://localhost:8080 in Chrome/Edge with WebGPU support
